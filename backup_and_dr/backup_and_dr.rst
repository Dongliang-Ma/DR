 Nutanix本地快照（时间流）
 .....................................

 用例
 
 - 防止客户操作系统损坏
 - 快照VM环境
 - 自助文件级还原

关键优势

 - VM为基本单元
 - 没有性能影响
-  有效降低存储节点的每GB成本

Nutanix Cloud Connect
.....................

用途
- 来自Nutanix的混合云解决方案
- 支持与Azure和AWS集成

用例

- 存档
- 备份

关键优势

 - 易于设置和管理
 - 广域网WAN加速传输
 - 与Nutanix DR产品天然兼容

.. figure:: images/backup_and_dr_03.png

异步复制
........................

用例

 - 防止VM损坏/删除
 - 防止完全站点故障

差异优势

 - 支持多站点DR拓扑
 - 同时支持多个保留策略共存
 - WAN广域网优化复制
 - Rest API驱动（客户DR Runbook）
 - VM和应用程序级别一致性

.. figure:: images/backup_and_dr_04.png

Metro双活高可用复制
.......................................

用例

 - 防止完全站点故障
 - 站点发生故障时不会丢失数据

关键优势

- 易于设置和管理
- 支持两地三中心（第3站点）
- 第三方仲裁选项
- Rest API驱动（客户DR Runbook）

跨管理程序备份和容灾
..............................

价值主张和收益

- 享受虚拟化管理程序自由选择的便利，同时使用内置集成备份和容灾服务
- 享受Nutanix异步远程复制技术的所有优点
- 自动迁移配置和VM转换

.. figure:: images/backup_and_dr_06.png

自助文件还原
.........................

用途

 - VM Snapshots中独立文件级别恢复
 - 无需恢复整个VM

差异优势

- 无需管理员干预的自助服务恢复
- 易于设置和管理
- 适用于中型客户和第2级工作负载的可行备份解决方案
